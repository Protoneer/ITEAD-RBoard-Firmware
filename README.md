ITEAD-RBoard-Firmware
=====================

Firmware to control the relays via serial connected to a XBee unit.
Commands are as follow:

#R1=0  --> Relay Off
#R1=1  --> Relay On
#R2=0  --> Relay Off
#R2=1  --> Relay On
#R3=0  --> Relay Off
#R3=1  --> Relay On
#R4=0  --> Relay Off
#R4=1  --> Relay On
#CLEARALL  --> Turn all off
#SETALL    --> Turn all on
